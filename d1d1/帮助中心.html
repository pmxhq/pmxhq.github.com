新手入门

Q : 如何注册第一配资网？

A : 进入第一配资网首页，点击右上角“注册”按钮。

     根据提示信息填写注册信息，获得短信验证码，免费注册。

Q : 如何登陆第一配资网？

A : 进入第一配资网首页，点击右上角“登录”按钮。

     在登录页面注册用户可以通过用户名、绑定的手机号、邮箱登录。

Q : 如何找回密码？

A : 如果您遗忘登陆密码，可以在网站登陆页面，点击“忘记密码”按钮，输入手机号码和手机短信验证码

     进行密码重置，就可以用新密码登陆了。

Q : 如何修改个人资料？

A : 用户登陆进入个人中心，在个人信息和安全中心可以查看并修改个人资料。

Q : 交易密码和登录密码一样吗？

A : 交易密码是提取现金的密码，不同于登陆密码，是用来保护资金安全的密码。

Q : 帐户及信息有哪些安全保障措施？

A : 登录第一配资网官方网址www.d1d1.com；

     给您的第一配资网帐户设置高强度登陆密码；

     注重电脑运行环境安全；

     注意保护个人隐私。

 

用户认证

Q : 为什么要进行实名认证？

A : 实名认证确认了用户的身份以及合同的有效性和合法性，同时也验证提现银行帐户属于您本人，确保您的资金安全。

     第一配资网对所有用户资料都将严格保密。

Q : 为什么要进行手机验证？

A : 手机验证可以提高帐户的安全级别，可以及时得到您交易账户的提醒信息。第一配资网对所有用户资料都将严格保密。

 

支付取现

Q : 第一配资网支持哪些支付方式？

A : 银行卡在线支付，支付宝支付、信用卡支付、线下支付。

Q : 如何开通网上银行？

A : 所有商业银行都有个人网银业务，你只要携带银行卡和有效身份证到当地发卡行营业网点，即可开通网上银行支付，转

     帐。办理网银U盾，使用更方便。

Q : 如何提取现金？

A : 登陆进入配资管理点击交易帐户里提现按钮，正确填写：金额、开户人、开户行（**银行**分行**支行）、银行帐号。
